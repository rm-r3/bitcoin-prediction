About This Project

How it works:

This tool uses a neural network trained on historical Bitcoin data (price, volume, date) to predict market sentiment categories: Extreme Fear, Fear, Neutral, Greed, and Extreme Greed.

Data Source: Historical Bitcoin prices and volumes (2018-2024)

Model: ML5.js Neural Network with classification

Disclaimer: This is a learning project. Cryptocurrency markets are highly volatile and unpredictable. Never use this for actual investment decisions. Always DYOR (Do Your Own Research)!
